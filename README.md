duojs-server
============

一个专门为[duojs](https://github.com/duojs/duo)开发的开发环境。

[component/wiki](https://github.com/componentjs/component/wiki/Components)

这块只是暂时托管，之后会写成Yeoman的Generator。这样就方便多了。

### server
起个服务

```
$ gulp server
```

### build
使用`duo.js`build

```
$ gulp build
```

### publish
压缩资源脚本

```
$ gulp publish
```
